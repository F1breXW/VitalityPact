# 快速开始指南

## 🎯 立即体验调试模式

### 方法1：右上角调试按钮（最简单）

1. 打开应用，点击"跳过授权"
2. 进入主界面后，你会看到顶部有一个蓝色提示横幅：
   > "连续点击右上角扳手图标5次打开调试面板"
3. 点击右上角的**扳手图标🔧**
4. 连续点击5次，你会看到数字显示：0/5 → 1/5 → 2/5 → 3/5 → 4/5 → 5/5
5. 第5次点击后，调试面板自动打开！

### 方法2：右下角按钮（备用）

- 连续点击屏幕右下角5次也能打开调试面板

---

## 🎮 调试面板功能

### 演示场景（推荐）
点击预设场景，一键体验：

1. **场景1: 虚弱状态** - 步数0，睡眠4小时
2. **场景2: 健康状态** - 步数6000，睡眠7小时
3. **场景3: 亚健康** - 步数8000，睡眠3小时
4. **场景4: 满状态** - 步数10000，睡眠8小时

### 手动调节
- 步数：0-15,000步
- 睡眠：0-12小时
- 运动：0-120分钟
- 心率：50-150 BPM

---

## 📱 界面说明

### 主界面元素
- **🪙 金币**：显示当前金币数量（步数/10）
- **状态指示**：显示角色当前状态（健康/疲劳/兴奋）
- **🔧 调试按钮**：点击5次打开调试面板
- **角色**：点击角色触发AI对话
- **数据面板**：显示步数、睡眠、运动进度

### 角色状态
- 😵‍💫 **疲劳**：睡眠不足，防御力减半
- 😊 **健康**：状态良好，随时准备冒险
- 🤩 **兴奋**：能量满溢，战斗力MAX

---

## 🎬 演示流程（5分钟）

1. **进入应用** → 跳过授权
2. **打开调试面板** → 右上角扳手图标点击5次
3. **选择场景** → 点击"场景3: 危机降临"
4. **观察变化** → 角色状态、AI对话、视觉反馈
5. **切换场景** → 尝试其他预设场景

## ❓ 常见问题

**Q: 找不到调试按钮？**
A: 看右上角状态栏，有一个🔧扳手图标

**Q: 点击没反应？**
A: 请连续快速点击5次，间隔不要超过3秒

**Q: 调试面板是空的？**
A: 先开启"调试模式"开关

**Q: 想重新看提示？**
A: 重启应用即可再次看到提示横幅
